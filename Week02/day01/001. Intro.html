Formatting Tags Introduction

'b' tag is used to get a bold text.
'i' tag is used to get an italic text.
'em' tag is also used to get italic text
'u' tag is used to get an underlined text.
'strike' tag is used to get a strikethrough text.
Hello'sup' tag is used to superscript a text
Hello'sub' tag is used to subscript a text
'ins' tag is used to get underlined text.
'big' tag is used to get a BIG text.
'strong' tag is used to get bold text.
q tag puts the text in quotation mark.
'mark' tag is used to hightlight text
'center' tag is used to center the text.

Some tags like big, center and strike are no longer allowed in HTML5 at all and they have been removed completely. (Deprecated)

'marquee' tag is used to get a moving text! Changing direction of 'marquee' tag to up. Changing direction of 'marquee' tag to right. Changing direction of 'marquee' tag to down. This is a hightlighted marquee tag.

By default marquee tag's direction is "left".

Types of HTML Tags

Paired/Container tags

Have an opening and closing tag.

Unpaired/Non Container tags

No closing tag.
